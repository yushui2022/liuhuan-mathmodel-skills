# 正式论文 Markdown 模板

Agent 写 `paper_output/final_paper_source.md` 时按此模板展开。不要机械保留占位符。

```markdown

{{针对问题一，说明模型、算法和关键结果。}}
```

## 写作注意

- 每个 `5.x` 必须引用该问的表格、图表、指标和结论。
- 图表不能只插入；必须解释图表揭示的规律。
- 如果某问没有真实结果，不得写成最终结论，必须回到建模代码补证据。
- 不要把 `paper_output/tasks.json` 的微单元标题机械拼接成正文。
